## 4. Git e GitHub: compartilhando e colaborando em projetos

Ness curso, aprendi com versionamento de código Git. Git e Github são duas coisas diferentes mas que se completam, quanto git é apenas uma forma de criar uma versão para cada código, Github armazena tais modificações em formato de repositório remoto. Dentre as atividades do curso: como commitar, adicionar um commit, verifica todos os commits anteriores, como verificar se um arquivo foi modificado ou adicionado para commit, ou até mesmo atualizado, como pegar commits de um repositório remoto, e até mesmo enviar os seus commits para o repositório.

Repositório: **/compartilhando-colaborando-projetos**
