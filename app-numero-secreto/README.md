# Jogo do Número Secreto (app-numero-secreto)

O jogo do número secreto é uma aplicação construida com HTML5, CSS3 e JavaScript. Tem como principal objetivo descobrir um números secreto com as dicas enviadas pelo programa, caso o usuário coloque um valor menor/maior que o número secreto, ele será orientado pelo programa até acertar os valores, o primeiro projeto (v1) utiliza prompts, enquanto o segundo projeto faz uso elementos do HTML.

## 2. Lógica de programação: mergulhe em programação com JavaScript.

A versão 01 vai apenas gerar um número aleatório, fazendo com que o usuário tenha chance de ganhar, contabilizando a quantidade de vezes que ele tentou até acertar.

## 3. Lógica de programação: explore funções e listas.

A versão 02 tem a mesma função que a primeira, a única diferença é manipulação com HTML/JavaScript, fazendo com que o usuário não aperte em "Reiniciar" antes de encontar tal número, quando ele erra é uma mensagem, e quando ganha também, não permitindo que ele gere o mesmo número aleatório, mas havendo um limite entre 1 e 5, e a cada jogo ganho, o valor não será mais o anterior.
